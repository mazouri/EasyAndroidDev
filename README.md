一、项目介绍

        EasyAndroid旨在搭建一个使开发人员快捷高效开发Android项目的框架，使用的是AndroidStudio开发。
        
        本人会根据自己在项目中的所学所用，以及学习开源大牛的优秀项目，逐渐集成到本项目中。

二、项目地址

        Github地址：https://github.com/mazouri/EasyAndroidDev
        CSDN博客主页：http://blog.csdn.net/dongdong230
